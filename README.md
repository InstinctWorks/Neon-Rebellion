# Neon Rebellion

Neon Rebellion is a solo-developed 2D bullet-heaven survival game built in Godot. Set in a cyberpunk-themed world, players fight against endless waves of hostile drones while surviving as long as possible.

The game features automatic attacks, enemy progression, XP collection, kill tracking, and upgrade choices that allow the player to grow stronger over time. As time progresses, stronger elite enemies appear, creating increasing pressure and difficulty.

## Features

* Endless wave survival gameplay
* Automatic combat system
* XP drop and level-up progression
* Upgrade selection system
* Health pickups for recovery
* Bomb pickups that clear all active enemies (excluding bosses)
* Pause and restart systems
* Elite boss encounters every 3 minutes
* Increasing difficulty over time (enemy health, speed, and damage scale after each elite boss defeat)

## Gameplay Systems

### XP and Progression

Enemies drop XP upon defeat:

* Yellow XP for regular enemies
* Orange XP for elite enemies

Players level up by collecting XP and can choose from upgrades such as:

* Increased Max Health
* Increased Damage
* Increased Movement Speed

### Boss Encounters

Every 3 minutes, an elite mechanical hound appears.

Boss encounters:

* Clear all active enemies on spawn
* Pause normal enemy spawning
* Trap the player inside a battle arena
* Feature charge-based attacks
* Increase enemy health, speed, and damage after each boss defeat

## Enemy Types

* Multiple drone enemy variants
* Elite mechanical hound boss
